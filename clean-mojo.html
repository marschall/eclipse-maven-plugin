eclipse:clean

Full name:

com.github.marschall:eclipse-maven-plugin:2.11.1:clean

Description:

Deletes the .project, .classpath, .wtpmodules files and .settings folder used by Eclipse.

Attributes:

  • Requires a Maven project to be executed.

Optional Parameters

Name Type Since Description
<additionalConfig> EclipseConfigFile[] - additional generic configuration files for eclipse
<basedir> File - The root directory of the project
User property is: basedir.
<packaging> String - Packaging for the current project.
User property is: project.packaging.
<skip> boolean - Skip the operation when true.
Default value is: false.
User property is: eclipse.skip.
